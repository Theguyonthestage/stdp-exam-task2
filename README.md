# STDP – Task 2 (Java)

This project solves Task 2 from the STDP exam.

The program:
- Creates a rectangular table with random size (2–10 rows and columns)
- Fills the table with 1s
- Asks the user to enter a row and column (1-based, with validation)
- Writes numbers 2, 3, 4, ... diagonally down-left until the table border
- Prints the table before and after modification
